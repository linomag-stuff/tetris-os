# tetris-os  
The first actual gaming focused linux distro, because you literally cant do anything else.  
7.3MB file size for the iso  
May be able to run in L3 cache of a high end Ryzen CPU (not tested)  
## How to use  
It features only 3 commands: vitetris, shutdown and reboot  
## Versions  
There are 2 main versions ready to download:  
TetrisOS (the actual one)  
TetrisOS-bare (its stripped down to less stuff)  
## Features  
TetrisOS: has a simple ">" shell and help commands  
TetrisOS-bare: Gets rid of the ">" shell and help commands  
## How To Boot  
USB using boot menu (not tested)  
QEMU with Virt-manager (tested)  
VMware (not tested)  
VirtualBox (not tested)  
## How to build from source  
The instructions are in INSTRUCTIONS file  
## Requirements  
Architecture: x86_64  
RAM: 64MiB minimum  
Storage: 7.3MB for the ISO  
Electricity: optional
